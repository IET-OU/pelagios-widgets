Pelagios Place widget - alpha version

Introduction

This is a widget that you can embed on a web page. The widget is associated with a particular place which you specify when you embed the widget. It provides a small Pelagios icon which can be clicked on to expand to give information about the place specified including the content from Pelagios partners associated with that place.
